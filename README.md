# SA Constitution Knowledge Graph
A machine-readable graph of South Africa's Chapter 1 (Founding Provisions) and Chapter 2 (Bill of Rights), extracted verbatim from official sources.

Use case: Fundamental Rights Impact Assessments (FRIA) for developers of high-risk systems (AI, surveillance, biometrics, automated decisions). Map system features directly to constitutional rights (privacy, equality, dignity, etc.) and check limitations against Section 36.

Data sources (official gov.za):

Chapter 1: https://www.gov.za/documents/constitution/chapter-1-founding-provisions

Chapter 2: https://www.gov.za/documents/constitution/chapter-2-bill-rights
